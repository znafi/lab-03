## Lab Description

This lab implements the ListyCity Android application with the following features:
- Display a list of cities with their provinces
- Add new cities using a FloatingActionButton
- Edit existing cities by clicking on them in the list
- Uses Fragments and DialogFragments for add/edit functionality
- Implements the City model with Serializable interface

## Implementation Details

- **City Class**: Model class implementing Serializable with name and province fields
- **AddCityFragment**: DialogFragment that handles both adding new cities and editing existing ones
- **MainActivity**: Main activity with ListView, ArrayAdapter, and click listeners for edit functionality
